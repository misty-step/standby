Seven complete app directions for a live meeting command surface.

Each prototype keeps the same product contract: visible capture health, newest-first transcript, an operator-controlled Ask Standby moment, approval before execution, live worker status, and honest speaker/audio attribution. The point is not palette swapping. Each screen organizes the work differently.

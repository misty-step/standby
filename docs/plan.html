Standby

Build a local-first meeting command surface that listens quietly, proposes concrete work cards, and routes approved cards to worker agents through a durable event ledger.

Outcome

Transcript spans produce a proposal card. Approval produces a completed mock research job and artifact.

Proof

./scripts/verify.sh runs Rust tests, frontend build, and an API smoke.

Stop

No external capture, live LLM, repo mutation worker, or hosted infra until the local loop is green.

Architecture Slice

Demo transcript source

Deterministic approval endpoint
Mock research worker emits normalized events
